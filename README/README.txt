Si es la primera vez que usa este aplicativo, seguramente no tendra ningun cliente ni pelicula ni sala registrada,
es por ello que al intentar editar el horario, le es imposible esta tarea, puesto que no hay peliculas ni salas.

Es por ello que para la experiencia minima usted debera de añadir primero peliculas y salas, sin embargo si quiere
la maxima experiencia que nuestra "compañia" ofrece, tambien puede inscribir los clientes. Con esto los clientes van 
tener un descuento aleatorio a la hora de comprar una pelicula, esto favorecera su negocio ya que hara que tenga una 
mayor retencion de clientes (Estudio realizado por compañia a la cual pagamos para hacerlo).

Una vez tenga tanto clientes, peliculas y sus salas registradas, usted querrera poder usar esta informacion para la 
proxima vez que vuelva a abrir el aplicativo, para esto usted tiene dos opciones, precionar manualmente el boton 
"Guardar cambios" en la pagina principal, sin embargo tambien si le da parece realizar un click para la proteccion de 
su informacion, basta con que cierre el programa cuando este en la pagina principal, si no lo hace en la pagina principal,
los datos no seran guardados (Grupo 2 S.A.S no se hace responsable por cualquier perdida de datos ocacionada por usted 
no estar guardando los datos en los momentos oportunos, o cerrar el aplicativo en una ventana donde no se guardarian los 
datos al cerrar).

Si usted desea contratar mas personas, usted debera de anadir la informacion de sus nuevos empleados a el documento de 
texto que guarda la informacion de estos trabajadores. Este documento de texto se encuentra en la misma carpeta donde se 
encuentra la carpeta que contiene este "README". Una vez en esta carpeta principal seguiras los siguientes pasos:
	-> Personas -> Trabajadores -> trabajadores.txt

Una vez en este documento de texto tendra que entrar los siguientes datos (en este mismo orden) separados por 3 barras ("---")
	nombre---fechaNacimiento(en formato dd/mm/aaaa)---numero telefono---codigo---salario

Asi el trabajador quedara registrado en el programa.
	
Cuando quiera cambiar el turno en el que se encuentra, solo tendra que presionar el boton "Turno Siguiente", el cual cambiara
cambiara el turno, tenga en cuenta que no se pueden comprar boletos a peliculas pertenecientes a un turno anterior al cual se 
encuentra, ademas de esto, si usted se encuentra en el ultimo turno (por defecto el turno 4), y vuelve a pasar al turno 1, 
se asumira como si usted a cambiado a un nuevo dia, por ende las salas se liberaran (es decir que todas las sillas pasaran a 
desocupadas).
